# hurling-match-data
Open dataset: puckout and set piece analysis, All-Ireland Senior Hurling Championship 2021–2025. Hand-coded from broadcast footage. Two event types, one row per event.

## Datasets
- **Puckouts** — goalkeeper restarts: type, delivery, zone, retention, next action
- **Frees & 65s** — set pieces: location, distance, shot type, side, pressure, result

## Coverage
- Competition: All-Ireland Senior Hurling Championship
- Seasons: 2021, 2022, 2023, 2024, 2025
- Scope: Knockout stages (semi-finals and finals; quarter-finals where available)
